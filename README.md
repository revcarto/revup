# RevUp™

Revenue Audit & GTM Blueprint — a RevCarto product.

- `/` — client-facing offer page (positioning, the 10 audit modules, deliverables, timeline, pricing, the RevPod™ retainer bridge)
- `/demo` — an interactive prototype of the internal RevUp Engine delivery platform, with RevCarto run through it as the first test workspace (composite scorecard, module findings, SaaS spend recovery sheet, 90-day fix list, GTM blueprint)

## Development

```sh
npm install
npm run dev
```

## Stack

Vite, React, TypeScript, Tailwind CSS, shadcn/ui components. Built to import cleanly into [Lovable](https://lovable.dev).
